For an UI to be intuitive to a user, especially first time users, it is important for the UI to be fimiliar to the user. This is why I have chosen to display the data as a simple line graph complemented with area shading to show the radio of active users and total users.

However, I do have to note that no amount of intuitive design is going to be equivilent to extensive user research. UX designers, ultimately, are not the end-users and therefore cannot design the UI to best fit the users' need. Only through extensive user research can you achieve the best usability.
